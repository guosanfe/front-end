### 什么是 Emmet？
==**Emmet**== 是代码编辑器的插件（VSCode、Sublime、WebStorm 等内置），通过类似 CSS 选择器的语法快速生成 HTML/XML 结构。

## 常用基础
### 1. 核心语法
| 语法            | 示例                        | 生成结果                                                                          | 说明                                        |
| ------------- | ------------------------- | ----------------------------------------------------------------------------- | ----------------------------------------- |
| **后代 `>`**    | `div>p`                   | `<div><p></p></div>`                                                          | 生成父子嵌套结构                                  |
| **兄弟 `+`**    | `div+p`                   | `<div></div><p></p>`                                                          | 生成平级元素                                    |
| **乘号 `*`**    | `li*3`                    | `<li></li><li></li><li></li>`                                                 | 重复生成多个元素                                  |
| **分组 `()`**   | `(div>p)*2`               | `<div><p></p></div><div><p></p></div>`                                        | 对一组元素进行重复或嵌套                              |
| **ID `#`**    | `div#header`              | `<div id="header"></div>`                                                     | 给元素添加 id 属性                               |
| **类 `.`**     | `div.box.red`             | `<div class="box red"></div>`                                                 | 给元素添加 class（可多个）                          |
| **属性 `[]`**   | `a[href="https://a.com"]` | `<a href="https://a.com"></a>`                                                | 添加自定义属性                                   |
| **文本 `{}`**   | `p{hello}`                | `<p>hello</p>`                                                                | 设置元素的文本内容                                 |
| **编号 `$`**    | `div.item$*3`             | `<div class="item1"></div><div class="item2"></div><div class="item3"></div>` | 生成递增数字，可配合 `$$` 补零（如 `item$$` → `item01`） |
| **自闭合标签 `:`** | `br` → `br` 或 `br/`       | `<br>`                                                                        | HTML5 中空元素默认无斜杠                           |
